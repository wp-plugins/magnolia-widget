== Description ==

A WordPress widget that displays Ma.gnolia bookmarks in yummy valid semantic XHTML code. The code is heavily commented and the output is in template functions so everything is fully customizable.

In this version, the plugin grabs the XML ma.gnolia feed and parses it with the PHP SimpleXML extension, which is included in PHP >= 5.0.

== Installation ==

1. Upload `magnolia-widget.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Activate the widget ('Ma.gnolia Widget') through the 'Widgets' tab under the 'Presentation' menu

== Options ==

* Username
* Number of Bookmarks
* Whether or not to show link to your Ma.gnolia profile
